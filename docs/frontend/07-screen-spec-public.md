# Opis ekranów publicznych

## 1. Strona główna
### Cel
Szybkie wejście do systemu i najważniejszych funkcji.

### Główne sekcje
- nagłówek z logo i przyciskiem logowania,
- hero section z polem do śledzenia przesyłki,
- sekcja skrótów do: utwórz przesyłkę, znajdź punkt, śledź przesyłkę,
- sekcja z krótkim opisem działania systemu,
- stopka.

### Główne akcje
- przejście do śledzenia,
- przejście do logowania,
- przejście do listy punktów.

## 2. Śledzenie przesyłki
### Cel
Sprawdzenie statusu przesyłki bez logowania.

### Elementy ekranu
- pole numeru przesyłki,
- przycisk „Szukaj”,
- karta aktualnego statusu,
- oś czasu zdarzeń,
- podstawowe informacje o przesyłce.

### Wyniki
- status aktualny,
- historia zdarzeń,
- przewidywany punkt odbioru / etap doręczenia.

## 3. Lista punktów i paczkomatów
### Cel
Wyszukanie miejsca nadania lub odbioru.

### Elementy ekranu
- wyszukiwarka miasta / kodu pocztowego,
- filtry typu punktu,
- lista wyników,
- karta punktu z adresem i godzinami,
- opcjonalny placeholder pod mapę.

## 4. Logowanie
### Cel
Wejście do systemu.

### Elementy ekranu
- logo i nagłówek,
- przycisk logowania przez zewnętrznego dostawcę,
- krótka informacja o rolach i dostępie po zalogowaniu.
