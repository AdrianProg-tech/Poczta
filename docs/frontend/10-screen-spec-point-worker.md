# Opis panelu pracownika punktu

## 1. Dashboard punktu
### Zawartość
- skróty do operacji punktowych,
- liczba przesyłek oczekujących,
- liczba przesyłek gotowych do wydania,
- szybkie przejście do przyjęcia i wydania przesyłki.

## 2. Przyjęcie przesyłki
### Zawartość
- pole numeru / skan,
- dane przesyłki,
- status płatności,
- potwierdzenie przyjęcia.

## 3. Wydanie przesyłki
### Zawartość
- numer przesyłki lub kod odbioru,
- dane odbiorcy,
- weryfikacja płatności / pobrania,
- potwierdzenie wydania.

## 4. Weryfikacja płatności
### Zawartość
- dane płatności,
- status,
- akcje: potwierdź / odrzuć / oznacz jako oczekujące.

## 5. Lista przesyłek w punkcie
### Zawartość
- przesyłki gotowe do odbioru,
- przesyłki nieodebrane,
- przesyłki w drodze do punktu,
- filtrowanie po statusie.
