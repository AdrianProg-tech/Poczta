# Opis panelu administratora

## 1. Dashboard administratora
### Zawartość
- podstawowe KPI,
- liczba przesyłek,
- liczba reklamacji,
- liczba płatności,
- szybkie linki do głównych modułów.

## 2. Użytkownicy
### Zawartość
- lista użytkowników,
- role,
- status aktywności,
- szczegóły użytkownika.

## 3. Punkty i paczkomaty
### Zawartość
- lista punktów,
- typ punktu,
- adres,
- status aktywności,
- edycja.

## 4. Przesyłki
### Zawartość
- pełna lista przesyłek,
- filtrowanie,
- szczegóły,
- historia zmian.

## 5. Płatności
### Zawartość
- lista płatności,
- statusy,
- płatności offline,
- akcje administracyjne.

## 6. Reklamacje
### Zawartość
- lista reklamacji,
- status,
- załączniki,
- decyzja administratora.

## 7. Raporty
### Zawartość
- statystyki przesyłek,
- statystyki płatności,
- skuteczność doręczeń,
- podstawowe wykresy i filtry.
