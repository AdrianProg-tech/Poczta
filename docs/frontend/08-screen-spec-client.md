# Opis panelu klienta

## 1. Dashboard klienta
### Zawartość
- skróty do najważniejszych akcji,
- liczba aktywnych przesyłek,
- ostatnie statusy,
- powiadomienia,
- lista ostatnich przesyłek.

## 2. Lista przesyłek
### Zawartość
- tabela lub karty przesyłek,
- numer przesyłki,
- status,
- typ dostawy,
- data utworzenia,
- akcje: szczegóły, opłać, przekieruj, reklamacja.

## 3. Szczegóły przesyłki
### Zawartość
- numer przesyłki,
- dane nadawcy i odbiorcy,
- aktualny status,
- status płatności,
- historia zdarzeń,
- punkt / adres dostawy,
- przyciski akcji.

## 4. Tworzenie przesyłki
### Układ
Wielokrokowy kreator:
1. nadawca,
2. odbiorca,
3. parametry paczki,
4. sposób dostawy,
5. usługi dodatkowe,
6. kto płaci,
7. podsumowanie.

## 5. Płatność
### Zawartość
- kwota,
- metoda płatności,
- przycisk rozpoczęcia płatności,
- status płatności.

## 6. Przekierowanie
### Zawartość
- aktualny punkt odbioru,
- lista dostępnych punktów,
- filtry,
- potwierdzenie nowego wyboru.

## 7. Reklamacje
### Zawartość
- formularz reklamacji,
- lista moich reklamacji,
- status sprawy,
- załączniki.

## 8. Profil użytkownika
### Zawartość
- dane użytkownika,
- zapisane adresy,
- ustawienia podstawowe,
- historia ostatnich aktywności.
