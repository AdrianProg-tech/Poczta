# Opis panelu kuriera

## 1. Dashboard kuriera
### Zawartość
- liczba dzisiejszych zadań,
- najbliższe doręczenia,
- szybkie przejście do listy zadań,
- alerty o nieudanych doręczeniach.

## 2. Lista zadań
### Zawartość
- numer przesyłki,
- odbiorca,
- adres,
- status,
- przycisk otwarcia szczegółów.

## 3. Szczegóły zadania
### Zawartość
- dane odbiorcy,
- adres,
- historia zdarzeń,
- planowane doręczenie,
- przyciski akcji: doręczono, próba doręczenia, awizo, błędny adres, zwrot.

## 4. Formularz akcji kurierskiej
### Zawartość
- typ akcji,
- komentarz,
- data/godzina,
- potwierdzenie zapisania zdarzenia.
